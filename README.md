# Quantum Drug Discovery

A research project exploring Hybrid Quantum-Classical Machine Learning for Blood-Brain Barrier (BBBP) prediction.

## Project Goal

Develop and compare classical machine learning and quantum machine learning models for predicting whether a molecule can cross the blood-brain barrier.

## Dataset

- BBBP Dataset (MoleculeNet)
